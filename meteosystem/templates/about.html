О Метеосистеме

Наш стек

Серверная часть погодного агрегатора написана на языке программирования Python (версия 3) и его высокоуровневом фреймворке Django (версия 5) с использованием системы управления базами данных MySQL. Для программной генерации форм применяется библиотека django-crispy-forms, а для поддержки междоменных запросов — django-cors-headers.

На клиенте работает «чистый» JavaScript (стандарт ES14), вся вёрстка построена на Bootstrap (версия 5).

Наша архитектура

Основой устройства приложения является вышеупомянутый Django, чей Model-View-Template шаблон проектирования был доработан добавлением служебного слоя «Services», поскольку целесообразно обособить объёмную логику отправки запроса и разбора полученного ответа метеорологической службы. Таким образом, была сформирована объектная модель домена. Это делает веб-приложение полностью соответствующим принципам Model-Controller-View, чего нельзя сказать о Django «из коробки».
